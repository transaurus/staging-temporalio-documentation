Interrupt a Workflow - Cancellation and Termination

Discover how Temporal enables you to gracefully handle Workflow interruptions through cancellations and terminations. Understand how to stop a Workflow cleanly with cancellation, allowing for proper cleanup and state management.

For situations where a Workflow is stuck, termination provides an immediate solution, ensuring your applications remain robust and responsive.
